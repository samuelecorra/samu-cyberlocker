# **Lezione 0 - Intro Unità Didattica 1 – Interfaccia del File System**

Questa unità introduce i **concetti fondamentali di file e directory**, descrivendo come il sistema operativo organizza i dati in modo logico e accessibile.  
Si analizzano le **caratteristiche dei file** (tipi, strutture, attributi) e la **struttura gerarchica del file system**, con l’obiettivo di comprendere come l’utente e i processi interagiscano con le risorse di memorizzazione tramite un’interfaccia **astratta, uniforme e indipendente dall’hardware**.
