# **Intro Unità Didattica 2 – Implementazione del File System**

Questa unità approfondisce la **struttura interna e l’organizzazione del file system**, analizzando come i file e le directory vengano **rappresentati, memorizzati e gestiti** a livello fisico.  
Verranno esaminate le **tecniche di realizzazione delle directory e dei file**, i **metodi di allocazione dello spazio su disco**, e gli strumenti per la **valutazione delle prestazioni**, al fine di comprendere come il sistema operativo traduca la visione logica dei file in operazioni concrete sui dispositivi di memoria.
