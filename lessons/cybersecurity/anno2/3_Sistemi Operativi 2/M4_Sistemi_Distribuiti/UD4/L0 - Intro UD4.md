# **Lezione 0 - Intro Unità Didattica 4 - File system distribuiti**

Questa unità introduce i **file system distribuiti (DFS)**, ovvero quei sistemi che permettono di **accedere a file e directory su più macchine** come se appartenessero a un’unica memoria logica.  
L’obiettivo è comprendere **come vengono gestiti, condivisi e sincronizzati i dati** in un ambiente distribuito, garantendo **trasparenza, coerenza e affidabilità** anche in presenza di nodi remoti o guasti di rete.
