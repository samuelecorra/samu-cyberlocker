# **Lezione 0 - Intro Unità Didattica 1 - Introduzione ai DBMS

Questa unità introduce il concetto di **Database Management System (DBMS)**, ossia il software che consente di **creare, gestire e interrogare** basi di dati in modo strutturato.  
L’obiettivo è comprendere **perché i DBMS hanno sostituito i tradizionali file system**, evidenziando i vantaggi in termini di **organizzazione, coerenza, sicurezza e indipendenza dei dati**.

Attraverso questa unità si analizzano le **caratteristiche principali**, i **servizi offerti** (come il controllo della concorrenza, il recupero da guasti, la sicurezza degli accessi) e le **funzionalità di alto livello** che rendono il DBMS un componente fondamentale di ogni sistema informativo moderno.

In sintesi, UD1 mostra come il passaggio dai file system ai DBMS rappresenti un’evoluzione cruciale: **dal semplice archivio di dati alla gestione intelligente dell’informazione**.
