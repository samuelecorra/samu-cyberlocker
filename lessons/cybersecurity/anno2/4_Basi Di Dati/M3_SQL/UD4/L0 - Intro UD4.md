# **Lezione 0 - Intro Unit? Didattica 4 - – DML per modifiche**

Questa unità affronta l’uso del **Data Manipulation Language (DML)** per le **operazioni di modifica dei dati** contenuti nelle tabelle.  
L’obiettivo è imparare a utilizzare i comandi che permettono di **inserire**, **aggiornare** e **cancellare** le tuple in modo controllato e coerente con i vincoli di integrità del database.

Si analizzeranno quindi i costrutti fondamentali:

- `INSERT` per l’inserimento di nuovi dati,
    
- `UPDATE` per la modifica di valori esistenti,
    
- `DELETE` per la rimozione di tuple.

Questi comandi rappresentano le **operazioni di base** di ogni linguaggio di gestione dei dati, indispensabili per mantenere aggiornato e coerente il contenuto delle basi di dati relazionali.
