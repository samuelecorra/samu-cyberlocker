# **Lezione 0 - Intro Unità Didattica 4 - Progettazione Logica**

Ci concentreremo ora sulla **trasformazione dello schema concettuale** (espresso tramite il modello Entità–Relazione) in uno **schema logico**, cioè una rappresentazione dei dati compatibile con il **modello del DBMS** che verrà utilizzato, tipicamente quello **relazionale**.  
In questa fase si applicano **regole di ristrutturazione e traduzione** per adattare lo schema alle esigenze del sistema, mantenendo inalterato il significato informativo.  
L’unità introduce inoltre le fasi successive di **documentazione**, **verifica di correttezza** e **progettazione fisica**, ponendo le basi per la realizzazione effettiva della base di dati.
