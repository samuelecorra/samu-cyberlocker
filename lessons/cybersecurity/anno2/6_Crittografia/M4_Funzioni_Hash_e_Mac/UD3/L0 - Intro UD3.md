# **Lezione 0 - Intro Unit? Didattica 3 - Funzioni MAC**


Introduciamo ora le **Message Authentication Code**, strumenti fondamentali per garantire **autenticità e integrità** dei messaggi attraverso una **chiave segreta condivisa**.  
In questa unità verranno analizzate le **proprietà principali** delle funzioni MAC, le **tecniche di attacco** più comuni e le **principali applicazioni pratiche** nei protocolli di sicurezza e nei sistemi di comunicazione.
